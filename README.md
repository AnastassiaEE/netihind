# Netihind.ee — Internet Packages Aggregator

A full‑stack web application that helps users find home internet packages available at a given address, built from scratch using modern technologies and best practices in **Next.js**.

🌍 **Live Site:** [https://www.netihind.ee](https://www.netihind.ee)  
📦 **Source Code:** [https://github.com/AnastassiaEE/Netihind](https://github.com/AnastassiaEE/Netihind)

---

## 🧠 Project Overview

Netihind.ee allows users to quickly discover, compare, and request internet packages based on their address. The app integrates multiple internet providers and presents options with sorting, filtering, and detailed package insights — making the process of choosing an ISP easier and more transparent.

Built entirely by me as a **solo full‑stack project**, this application demonstrates real‑world application architecture, performance optimization, and user‑focused design.

---

## ⭐ Key Features

- 🔍 **Search by Address:** Get all available internet packages for a specific location.
- 📊 **Sorting Options:** Sort results by speed or price.
- 🧰 **Advanced Filtering:** Filter packages by provider and connection technology.
- 📄 **Package Details:** See speed, price, discounts, required devices, and other metadata.
- ✉️ **Connection Requests:** Users can submit a request for connection directly through the platform.
- 🌐 **Localization (i18n):** Supports multiple languages.
- 📱 **Responsive Design:** Works smoothly on desktop and mobile screens.

---

## 🧰 Tech Stack

- 🚀 **Next.js (App Router)** – Full‑stack React framework with **SSR/SSG/ISR** support and **Server Components**
- 🎨 **Tailwind CSS v4** – Utility‑first styling
- 🗃️ **Supabase (PostgreSQL)** – Backend database and auth
- ⚛️ **SWR** – Data fetching with caching and revalidation
- 📧 **Resend API** – Sending emails (e.g., request confirmations)
- 🔁 **URL‑Synced State** – Filters/sorting reflected in the browser URL
- 📝 **WordPress (Headless CMS + GraphQL)** – Manages static pages in Markdown format

## 🚀 Getting Started

1. **Clone the repository**

```bash
git clone https://github.com/AnastassiaEE/Netihind.git
cd Netihind
```

2. **Install dependencies**

```bash
npm install
# or
yarn install
# or
pnpm install
```

3. **Start development server**

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

4. **Open your browser to:**
   🔗 [http://localhost:3000](http://localhost:3000)

Start editing pages in the app/ directory — changes hot-reload automatically.
